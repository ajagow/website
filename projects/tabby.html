tabby.

spring 2018

I am a known serial tab opener. I have this really bad habit of opening a whole bunch of tabs and then never closing them. Rather than change my behavior and start closing tabs more frequently, I decided to create a chrome extension that would help me keep better tabs (pardon the pun) on my tabs.

process.

first steps. Since I didn't know how to make a chrome extension, I followed a tutorial on how to set up the code for an extension.


features. Main features I wanted to be able to support were:

  • seeing all the tabs and in what windows they were in
  • see which tabs were playing audio
  • rearrange tabs from within the chrome extension via a drag and drop feature
  • view and be able to reopen the most recently closed tabs

easy animator.

fall 2017

A half-semester long project for Object-Oriented Design where I worked with Amy Luo to create an "easy animator."

The Java application is able to read in text files and render the specified animation either as a text description, as a SVG or in a JSwing window.

process.

paired programming. This was an exercise in coding as well as an exercise in team work. Amy and I coded the entirety of this project together; we were not allowed to work on any part separately. That being the case, we had to be able to communicate our ideas clearly and listen to each other in order to work effectively.


MVC. We followed the Model View Controller(MVC) architecture to create this project. The model included the shapes and animation objects and we had several different views (SVG, text, and Swing). The controller mediated the communication between the model and the view.


testing. All of our methods were tested with the JUnit library.

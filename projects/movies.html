movies and youtube.

fall 2018

A project for my DS4100 (Data Collection, Integration and Analysis) class where I tried to see if there is a connection between Youtube trailer views and movie revenue.

I was tasked with collecting and cleaning information and then building and evaluating predictive models.

You can watch me talk about my project in a 10 minute Youtube video or continue scrolling to find out more!

tools.

Python
R
TMDB api

process.

idea. One of my hobbies is watching movie trailers on Youtube so I thought it would be cool to incorporate that into my project.


data collection and cleaning. I used TMDB (the movie database) API to get data on the top 200 movies currently trending. In order to get the view count on Youtube, I had to write a python script that manually scraped the view count for each movie. I then did some basic cleaning to make sure all the data followed the same format.


building the models. In R Studio, I built a few different multiple linear regression models to try and create a model that had the best R-Squared.


conclusion. There was a very weak correlation (.32) between youtube trailer views and revenue. Instead, I found that budget and number of votes were better predictors for a movie's revenue.
